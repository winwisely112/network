# goAdmin

A Dashboard in go.

We can use this to give users ( in our multi tenant cloud) access to resources.


- security
	- has rbac
	- Has non captcha security Check to stop bots :)
- i18n
- 100% golang, so easy.
- plugin model, so easy to extend
- docker, so we can run on k8
